# Android 术语词典

本词典记录 `android` 目录内项目自有标识符的中文命名。Android、Kotlin、Compose、Room、Gradle 等外部框架 API 保持原名。

## 领域对象

| 原命名 | 中文命名 | 说明 |
| --- | --- | --- |
| Message | 帖子 | 用户在 BBS 中发布的一条内容 |
| MessagePayload | 帖子载荷 | 通过 BLE 传输的帖子数据 |
| WirePacket | 传输包 | BLE 传输层上的消息封装 |
| PacketMessage | 帖子包 | 携带帖子载荷的传输包 |
| Ack | 确认包 | 预留的确认传输包 |
| id | 编号 | 帖子或确认包的唯一编号 |
| text | 正文 | 帖子的文本内容 |
| sender | 发帖人 | 发出帖子的本机设备编号 |
| timestamp | 时间戳 | 帖子创建时间 |

## 存储

| 原命名 | 中文命名 | 说明 |
| --- | --- | --- |
| PostEntity | 帖子实体 | Room 中保存帖子记录的实体；因 Room/KSP 代码生成限制，源码类名暂保留英文 |
| PostDao | 帖子访问对象 | 帖子表访问接口；因 Room/KSP 代码生成限制，源码接口名暂保留英文 |
| AppDatabase | 应用数据库 | Room 数据库入口；因 Room/KSP 代码生成限制，源码类名暂保留英文 |
| upsert | 保存或更新 | 插入或替换帖子记录；因 Room/KSP 代码生成限制，源码方法名暂保留英文 |
| getAllLatestFirst | 按最新优先取全部 | 读取全部帖子，最新在前；因 Room/KSP 代码生成限制，源码方法名暂保留英文 |
| getAllLatestFirstFlow | 按最新优先监听全部 | 监听帖子列表变化；因 Room/KSP 代码生成限制，源码方法名暂保留英文 |

## 蓝牙

| 原命名 | 中文命名 | 说明 |
| --- | --- | --- |
| BleEngine | 蓝牙引擎 | 协调扫描、广播、连接、收发、存储 |
| BleScanner | 蓝牙扫描器 | 扫描附近同类设备 |
| BleAdvertiser | 蓝牙广播器 | 广播本应用的 BLE 服务 |
| BleGattServer | 蓝牙Gatt服务端 | 接收其他设备写入的数据 |
| BleCentralConnector | 蓝牙中心连接器 | 作为中心设备连接并写入对端 |
| BleConstants | 蓝牙常量 | BLE 服务、特征、厂商标记等常量 |
| BleLifecycleState | 蓝牙生命周期状态 | 蓝牙引擎当前状态 |
| BleLifecycleListener | 蓝牙生命周期监听器 | 向界面报告蓝牙状态变化 |
| PeerSnapshot | 邻机快照 | 当前连接、可写、待连接邻机数量 |
| TAG | 日志标记 | Android 日志调用使用的标记常量 |
| context | 上下文 | Android Context 实例；Room/KSP 相关源码中保留英文 |
| SERVICE_UUID | 服务UUID | 本应用的 BLE 服务 UUID |
| WRITE_UUID | 写入UUID | 用于写入帖子数据的特征 UUID |
| NOTIFY_UUID | 通知UUID | 预留通知特征 UUID |
| MANUFACTURER_ID | 厂商编号 | BLE 广播中的厂商数据编号 |
| APP_MARKER | 应用标记 | 区分本应用设备的广播标记 |
| onWriteReady | on可写就绪 | 中心端发现可写特征后的回调 |
| bluetoothAdapter | 蓝牙适配器 | Android 蓝牙适配器实例 |
| sendMessageToAllPeers | 发送帖子给所有邻机 | 将帖子编码后发送给所有可写邻机 |
| retrySendToAllPeers | 重试发送给所有邻机 | 使用已编码数据重试发送 |
| forceSync | 强制同步 | 手动向可写邻机补发历史帖子 |
| rearmScan | 重启扫描 | 重新启动扫描和广播 |
